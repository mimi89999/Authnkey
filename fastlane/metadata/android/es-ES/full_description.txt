Authnkey es un proveedor de credenciales que te permite usar llaves de seguridad FIDO2 (como YubiKey, SoloKey, Nitrokey o Token2) para autenticar llaves de acceso (passkeys) en Android.

El soporte integrado de WebAuthn en Android tiene una limitación importante: no es compatible con CTAP2 a través de NFC. Esto significa que no puedes usar llaves de acceso (passkeys) protegidas por un PIN ni credenciales residentes con llaves de seguridad NFC; estás limitado a usar la llave solo como un segundo paso básico para iniciar sesión. Authnkey soluciona esta limitación implementando el protocolo CTAP2 completo. 

La aplicación se registra como un proveedor de credenciales, por lo que cualquier aplicación o navegador que sea compatible con la API "Android Credential Manager" ofrecerá Authnkey como opción cuando se soliciten llaves de acceso (passkeys).

Esto también funciona en dispositivos sin Google Play Services, ya que Authnkey implementa el protocolo CTAP2 directamente, en lugar de depender de la API "FIDO2" de Google.

Funciones soportadas:
- Creación y autenticación de llaves de acceso (passkeys)
- Verificación con PIN (admite letras, números y símbolos)
- Permite iniciar sesión sin escribir tu usuario con credenciales residentes
- Soporta múltiples cuentas por sitio
- Es compatible con llaves de seguridad físicas NFC y USB

No requiere root.
